    Detailed Description

    This class acts as the main window for the print dialog.

    [protected] void QCpdWindow::closeEvent(QCloseEvent * event)

    This function overrides the closeEvent function from QMainWindow and is called when event occurs i.e. the user clicks on the window close (cross) icon in the window.

    [protected] void QCpdWindow::resizeEvent(QResizeEvent * event)

    This function overrides the resizeEvent function from QMainWindow and resizes all the child widgets to match the size of the resized layout items when event occurs.
