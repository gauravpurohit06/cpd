    Detailed Description

    The QQmlWidget class creates a widget from the qml files

    The dialog is comprised of three widgets - QQmlWidget, QCpdPreviewWidget and QPreviewToolbarWidget. QQmlWidget creates a QQuickWidget with pages/main.qml as the root and it handles the signals that are emitted from the pages/main.qml

    Member Function Documentation

    QQmlWidget::QQmlWidget(QWidget * parent = Q_NULLPTR)

    Constructs QQmlWidget objects with parent. Connects signals from pages/main.qml with slots in QQmlWidget.

    QQmlWidget::~QQmlWidget()

    Destroys QQmlWidget objects

    [slot] void QQmlWidget::cancelJob(int jobIndex)

    This function acts as a slot for the cancelJob signal emitted when the user selects "Cancel" menu item in the right click menu of the Jobs tab. The signal comes with jobIndex which is used to cancel the chosen job.

    [slot] void QQmlWidget::cpdQuit()

    This function acts as a slot for the signal cancelButtonClicked emitted from the pages/main.qml file. The signal is emitted when the user clicks on the "Cancel" Button in the sidebar.

    [slot] void QQmlWidget::printDocument(QString printerName)

    printDocument takes the name of the printer as a parameter printerName, looks up the PrinterObj whose name matches the parameter, sets printer settings to those chosen by the user and sends the file to printing.

    void QQmlWidget::resize(const QRect & rect)

    resize() takes a QRect& rect as a parameter and uses it to resize the QQmlWidget objects to the same dimensions as the parameter rect

    [slot] void QQmlWidget::setAdvancedOptions(QString printerName)

    This function finds the more advanced options like "Resolution", "Contrast", "Brightness" etc. for a given printer with name printerName and puts them in their corresponding QStringLists.

    Note: Due to some limitations with the CUPS API, only "Resolution" options are being set. This function will be updated as soon as we find a way to overcome the limitations in the API

    [slot] void QQmlWidget::setJobsHoldOptions(QString printerName)

    Many printers come with a hold option which specifies when the job should start after a user clicked the "Print" button in the UI. This function gets all the available options for a printer with the given name printerName and puts them in a QStringList called jobHoldOptionList.

    [slot] void QQmlWidget::setJobsList(bool activeOnly)

    The function gets the a list of all the jobs from all the backends. It then iterates over each job and concatenates the printer's name, job's user who started the job and the current state of the job and appends the string to a QStringList called jobsList.

    When activeOnly is set to 1, only jobs are that are currently running are shown whereas when it is set to 0, all jobs are shown.

    The "%" parameter is used a separator to split the three strings: printer's name, job's user, job's state.

    [slot] void QQmlWidget::setResolutionSetting(QString resolutionValue, QString printerName)

    This function acts a slot for the resolutionValueChanged signal emitted from the main.qml. It takes the name of the printer printerName and the resolution value selected by the user as resolutionValue and add it to the printerObj's settings struct.
