    Detailed Description

    The QQmlWidget class creates a widget from the qml files

    The dialog is comprised of three widgets - QQmlWidget, QCpdPreviewWidget and QPreviewToolbarWidget. QQmlWidget creates a QQuickWidget with pages/main.qml as the root and it handles the signals that are emitted from the pages/main.qml
